# Différents opérateurs

**Il existe différents opérateurs de comparaison :**

| Opérateur |      Description       |
| :-------: | :--------------------: |
|    ==     |        Égalité         |
|    !=     |       Différence       |
|     >     |  Stricte supériorité   |
|     <     |  Stricte infériorité   |
|    >=     | Supériorité ou égalité |
|    <=     | Infériorité ou égalité |
